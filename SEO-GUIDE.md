## 📝 Optimisation On-Page

### Balises Meta
Déjà optimisées dans chaque fichier HTML :
- Title : 50-60 caractères
- Description : 150-160 caractères
- Keywords : 5-10 mots-clés pertinents

### Structure HTML5
Utilisation correcte des balises sémantiques :
- `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`
- `<h1>` unique par page
- Hiérarchie `<h2>` → `<h3>` → `<h4>`

### Images
- Attributs `alt` descriptifs
- Noms de fichiers optimisés (ex: rose-classique-points-croix.jpg)
- Compression pour chargement rapide
- Tailles appropriées (800x600px recommandé)
